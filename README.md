- **Tên dự án: Website trao đổi hàng hóa** (dựa trên ứng dụng Chợ tốt)
- Công nghệ sử dụng
    - Front-end (admin page): 
        - ReactJS
        - react-router-dom
        - @mui/material
        - @mui/material-icons
    - Front-end (main page | customer page): 
        - ReactJS
        - react-router-dom
        - @mui/material
        - @mui/material-icons
